# Astronaut Color Puzzle

Astronaut Color Puzzle — это 3D-игра-головоломка на Unity, в которой игрок управляет космонавтом и использует переключение цвета энергии для прохождения уровня и активации различных механизмов.

## Функционал игры

В игре реализованы следующие основные механики:

- Переключение цвета скафандра между жёлтым, синим и фиолетовым.
- Цветные двери, которые открываются только при совпадении цвета игрока с цветом двери.
- Цветные кнопки и панели, активирующие объекты уровня: мосты, проходы и отключение лазеров.
- Переносимые цветные кубы, которые можно поднимать, переносить и устанавливать на пьедесталы для запуска механизмов.
- Зона энергетической волны, где игрок должен быстро переключаться на нужный цвет, чтобы избежать смерти и пройти испытание.
- Система чекпоинтов, сохраняющая точку возрождения игрока.
- Возрождение игрока после смерти от препятствий, например от лазера или после неудачного прохождения опасной зоны.

## Описание игрового процесса

Игрок проходит уровень последовательно, используя цветовые механики для решения головоломок.  
Сначала необходимо подобрать правильный цвет для открытия двери, затем активировать панель для появления моста, отключить лазер с помощью другой кнопки, пройти опасную зону с энергетическими волнами, после чего использовать переносимый куб для активации финального механизма и открыть путь к завершению уровня.

Игра построена на сочетании простых механик, которые постепенно комбинируются и усложняют прохождение.
